Zamboni

A Discord bot for tracking NHL player stats, team stats, league leaders, and draft history.

Add to Discord Vote on top.gg

Slash Commands

  • /seasonstats [Player] - Current season stats
  • /careerstats [Player] - Career stats
  • /playerinfo [Player] - Player info
  • /teaminfo [Team] - Team info
  • /teamroster [Team] - Full roster
  • /seasonleagueleaders - Current league leaders
  • /draftselections [Year] [Start] [End] - Draft picks
  • /help - Full command list
