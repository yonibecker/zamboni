Privacy Policy

Last updated: April 1, 2026

1. What Data We Collect

Zamboni does not collect, store, or log any personal data. Specifically:

  • We do not store Discord usernames, user IDs, or message content.
  • We do not store server names, server IDs, or channel information.
  • We do not track command usage or maintain any analytics.
  • We do not use cookies or any tracking technologies.

2. How the Bot Works

When you use a slash command, Zamboni receives the command and its parameters from Discord, fetches the requested NHL data from public APIs, and returns the result. No part of this interaction is stored or logged beyond what is needed to process the request in real time.

3. Third-Party Services

Zamboni fetches data from the following third-party APIs:

  • NHL API (api-web.nhle.com, api.nhle.com) - Player, team, and league statistics
  • NHL Records (records.nhl.com) - Draft history

These services have their own privacy policies. Zamboni does not send any user data to these APIs — only NHL-related queries (e.g., player names, team names, season years).

4. Data Shared with Discord

Zamboni operates within Discord's platform and is subject to Discord's Privacy Policy. The bot only accesses data that Discord provides through its API for slash command interactions.

5. Children's Privacy

Zamboni does not knowingly collect any data from users of any age, as it does not collect data at all.

6. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected on this page with an updated date.

7. Contact

If you have questions about this privacy policy, you can reach out via the bot's GitHub repository.
